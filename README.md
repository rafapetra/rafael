# Rafael Petrachini - Portfolio

#### By _Rafael Petrachini_

Website showing Rafael's portfolio with links to his projects.\
https://rafapetra.github.io/rafael

## Technologies Used

* _HTML_
* _CSS_

## Description

_This website displays a menu with links to 3 projects from Rafael's portfolio, as well as an "About Me" section with background information, hobbies and education and a Contact Me session._

## Setup/Installation Requirements

1. Clone this repository to your desktop.
2. Navigate to the top level of the directory.
3. Open index.html in your browser.
